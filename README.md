## Github User Search
This is a simple application search and display Github User data

### Tools Used
- React
- React-Redux
- reactstrap
- react-bootstrap-table-next
- react-toastify
- font-awesome
- node-sass
- eslint

### Setup Requirements
- node at least v10
- npm
- Your favourite browser

### Getting Started
- Clone this repo
- Install the dependencies, `npm install`
- start the development server, `npm start`
