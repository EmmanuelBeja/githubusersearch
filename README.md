## Search
This is a simple application search and display data

### Tools Used
- React
- React-Redux
- reactstrap
- framer-motion
- font-awesome
- node-sass
- react-tooltip
- eslint

### Setup Requirements
- node at least v10
- npm
- Your favourite browser

### Getting Started
- Clone this repo
- Install the dependencies, `npm install`
- start the development server, `npm start`
